fblib
==========

A simple Facebook library.

Installing
----------

Best way to isntall this package is by running goinstall:

	go get robteix.com/v2/fblib

And then you can import it in your code like this:

	import (
		...
		"robteix.com/v2/fblib"
		...
	)
